## Technical Specifications for All Images

### File Naming Convention:
- Thumbnails: `[case]_thumb.png` (e.g., `partnerships_thumb.png`, `sap_thumb.png`)
- Internal Images: `[case]/situation.jpg`, `[case]/task.jpg`, `[case]/action.jpg`, `[case]/result.jpg`

### Resolution Requirements:
- Thumbnails: 800x500px minimum (16:10 ratio)
- Internal Images: 1920x1080px minimum (16:9 ratio)
- File Size: Optimized for web, <500KB per image ideal

### Accessibility:
- Sufficient contrast ratios (4.5:1 minimum)
- Clear visual hierarchy
- No critical information in color alone
- Clean, readable at multiple sizes

### Brand Consistency:
- Google Material Design principles throughout
- Consistent color palette across all images
- Roboto font family exclusively
- 25-30% minimum whitespace
- Professional, executive-appropriate aesthetic
